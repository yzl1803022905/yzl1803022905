Ubuntu 基础指令

一，增删移复寻装

增

新建文件：touch (文件名)

新建文件夹： mkdir (文件夹名)

（注：vim 可直接新建文件）

删

mkdir (目录名) 创建一个目录
rmdir (空目录名) 删除一个空目录
rm (文件名) 删除一个文件或多个文件
rm -rf (非空目录名) 删除一个非空目录下的一切 (不可逆)

感兴趣的可以试试对自己的虚拟机使用以下指令 rm -rf /;

千万别对本机用，出了事概不负责。（手动狗头）

移动

mv （路经/文件 /经/文件移动相对路经下的文件到绝对路经下）
mv （文件名 （新名称 在当前目录下改名））

复制

cp （路径/文件 路径/（可以修改）（文件名））

寻找

ls (普通的列出)

ls -a(列出所有文件并且包含隐藏文件 (前方加.) )（注意，不是解析出文件中的其他文件，无法破解隐写术，若要破解隐写术请使用Binwalk）

ls -F （在可执行文件后加* 目录后加/ 链接后加@ ）

ll (列出文件表明日期 有后缀名)

cat （文本文件） 直接查看

cat –n （文件名） 主要是-n可以显示行号

more （文件名） 类似cat enter下一页 q退出

less /路径/con.cfg （类似于more，文本阅读指令，less指令执行完后会有一个：，如果想向下翻页按空格键，如果想向上翻页按B键，如果想搜索一个字符串可以用/字符串来搜索（vim中操作类似），最后按q键退出）

grep “字符串” 文件名 （于文件中查找）

装（apt）

apt-cache search package #搜索包
apt-cache show package #获取包的相关信息，如说明、大小、版本等
sudo apt-get install package #安装包
sudo apt-get install package - - reinstall #重新安装包
sudo apt-get -f install #修复安装”-f = –fix-missing”
sudo apt-get remove package #删除包
sudo apt-get remove package - - purge #删除包，包括删除配置文件等
sudo apt-get update #更新源
sudo apt-get upgrade #更新已安装的包
sudo apt-get dist-upgrade #升级系统
sudo apt-get dselect-upgrade #使用 dselect 升级
apt-cache depends package #了解使用依赖
apt-cache rdepends package #是查看该包被哪些包依赖
sudo apt-get build-dep package #安装相关的编译环境
apt-get source package #下载该包的源代码
sudo apt-get clean && sudo apt-get autoclean #清理无用的包
sudo apt-get check #检查是否有损坏的依赖
sudo apt-get clean #清理所有软件缓存（即缓存在/var/cache/apt/archives目录里的deb包）

用数字表示rwx的权限

其中a,b,c各为一个数字，a表示User，b表示Group，c表示Other的权限。

r=4，w=2，x=1
- 若要rwx（可读、可写、可执行）属性，则4+2+1=7
- 若要rw-（可读、可写、不可执行）属性，则4+2=6
- 若要r-w（可读、不可写、可执行）属性，则4+1=5

vim echo_study.sh

chmod 777 echo_study.sh （是文件可执行）
